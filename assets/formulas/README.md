## assets/formulas

JSON definitions for all formulas grouped by domain.

- `energy_band_structure.json` — Energy band and dispersion formulas.
- `density_of_states_statistics.json` — DOS and semiconductor statistics formulas.
- `carrier_concentration_equilibrium.json` — Carrier concentration and equilibrium relations.
- `carrier_transport_fundamentals.json` — Carrier transport (drift/diffusion) formulas.
- Additional files may define other categories (e.g., PN junction) following the same pattern.
