## lib/ui

Flutter UI code for the app.

- `controllers/` — Business-logic controllers for panels and interactions (e.g., `formula_panel_controller.dart`).
- `widgets/` — Reusable UI components (formula cards, inputs, result cards, LaTeX text, theme).
- `pages/` — Top-level screens/pages.
- `theme/` or `formula_ui_theme.dart` — Shared styles and typography helpers.
- `latex_text.dart` — Renders LaTeX strings in the UI.
